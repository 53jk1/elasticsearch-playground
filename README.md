# Elasticsearch playground

This repo was created to fastly run an Elasticsearch cluster with Kibana and Logstash.

## Requirements

- Docker
- Docker Compose

## How to run

```bash
docker-compose up -d
```

## How to stop

```bash
docker-compose down
```

## How to access

- Elasticsearch: http://localhost:9200
- Kibana: http://localhost:5601
- Logstash: http://localhost:9600
